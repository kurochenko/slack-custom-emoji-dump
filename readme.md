# Slack Custom Emoji Crawler

### Prerequisites:
- python 3.5+
- Slack OAuth token

## Running script 
### (optional) create virtualenv
`python3 -m venv .env && . .env/bin/activate`
### Install dependencies
`pip install -r requirements.txt`

### Execute the script
`python3 get_emojis.py AUTH_TOKEN`
